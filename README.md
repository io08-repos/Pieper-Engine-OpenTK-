# Pieper Engine (OpenTK)

Pieper Engine is a custom 2D graphics and physics engine built using OpenTK and OpenGL.

The project was created to explore how rendering pipelines, entity systems, and collision detection can be implemented from scratch without relying on a full game engine.

## Features

- Rendering pipeline using OpenGL
- Mesh and material system for renderable objects
- Camera system using model, view, and projection matrices
- Collision detection using the Separating Axis Theorem (SAT)
- Entity Component System (ECS) architecture

## Technologies

- C#
- GLSL shaders
- OpenTK
- OpenGL

## Project Goals

The goal of the engine is to experiment with graphics programming and systems architecture, and to better understand how game engines structure rendering, physics, and entity systems.

Future development will focus on:
- Expanding the rendering system
- Improving the ECS architecture
- Including a scene editor environment
